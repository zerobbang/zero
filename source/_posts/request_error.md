```python
[Errno Expecting value] Too many API requests.: 0
```

**요청 수가 특정 시간 동안 처리 될 수 있는 속도를 초과할 때 발생하는 문제라고 한다.**  

이를 해결하기 위해 time.sleep으로 제어를 해주거나 try문으로 제어를 해줄 수 있다.
